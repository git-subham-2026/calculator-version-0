# Python Calculator

A simple terminal-based calculator made with Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Operator validation
- Division by zero handling

## How It Works
1. User enters two numbers
2. User selects an operator
3. Program performs calculation
4. Result is displayed in terminal

## Example
Enter your first number : 12
Enter your second number : 4
Enter operator such as '+', '-', '*', '/' : /

12 / 4 = 3.0

## Run
Make sure Python is installed.

Run:
python main.py
